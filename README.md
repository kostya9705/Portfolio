# Portfolio

Портфолио по задачам в области анализа данных.

## Проекты

### 1. TelecomCustomerAnalysis
Анализ клиентов телеком оператора с фокусом на отток (churn analysis).

**Задачи:**
- Определение факторов, влияющих на отток клиентов
- Сегментация пользователей по демографическим и поведенческим признакам
- Визуализация ключевых метрик и закономерностей

**Стек:** pandas, numpy, matplotlib, plotly, seaborn, scipy

### 2. TelecomSMSAnalysis
Анализ временных рядов СМС-трафика с интервалом 15 минут.

**Задачи:**
- Декомпозиция временного ряда (тренд, сезонность, остатки)
- Выявление паттернов и аномалий в трафике
- Подготовка данных для прогнозирования

**Стек:** pandas, matplotlib, statsmodels, seaborn
